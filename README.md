# IFF Yii-extensions
yii addons: classes, components, patches

Классы, компоненты и иные дополнения для yii (в т.ч. yii2)

## Установка (Usege)

Установка с помощью composer:

```
composer require it-for-free/yii-extensions:~v0.0.1
```
(install via composer).
